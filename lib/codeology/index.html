Codeology

Codeology is an open-source project that brings to life the art and science of code. An algorithm analyzes Github projects and creates unique organic forms based on the codebase size and language. As no two pieces of code are alike, no two Codeology forms are alike.

      What it is

      Codeology is an open-source project that reimagines code by bringing it to life visually. The combination of coding languages in each Github project determines its look, therefore, each creation is original.
